# Second Life — Marketplace Barang Bekas

🌐 **Live Demo:** [https://uap-klompok-10.vercel.app/](https://uap-klompok-10.vercel.app/)

Marketplace barang bekas berbasis web yang mendukung konsumsi bertanggung jawab sesuai SDG 12.

---

## Fitur

| # | Fitur | Deskripsi |
|---|-------|-----------|
| 1 | Listing & Storytelling | Tampilkan barang dengan cerita di baliknya |
| 2 | Search & Filter | Cari berdasarkan nama, harga, dan kondisi |
| 3 | Impact Tracker | Hitung dampak lingkungan dari transaksi barang bekas |
| 4 | Form Jual | Simulasi listing barang baru dengan preview real-time |

## Struktur File

```
UAP/
├── index.html        # Versi terpisah (HTML + link ke CSS & JS)
├── style.css         # Stylesheet
├── script.js         # JavaScript
└── fonts/            # Font Guton (local)
    ├── Guton-Regular.otf
    ├── Guton-Medium.otf
    ├── Guton-SemiBold.otf
    ├── Guton-Bold.otf
    ├── Guton-ExtraBold.otf
    └── Guton-Black.otf
```

## Font

- **Guton** — heading, logo, angka statistik (lokal, `fonts/`)
- **Poppins** — body, tombol, form (Google Fonts)

## Teknologi

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Poppins)
- Deployed via [Vercel](https://vercel.com)
